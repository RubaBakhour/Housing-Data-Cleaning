# Housing Data Cleaning Using SQL Server

This project focuses on cleaning and transforming a raw housing dataset using SQL Server. The goal is to prepare inaccurate, inconsistent, and unstructured real estate data into a clean, analysis-ready format following real-world data preparation standards.

This project demonstrates key data wrangling, data quality, and SQL transformation skills.

---

## Project Objectives
- Standardize inconsistent date formats  
- Handle missing property address values  
- Split complex address fields into structured columns  
- Normalize categorical values  
- Detect and prepare duplicate records  
- Remove unused and redundant columns
